# Frontend (Vue3 + Vite)

## 1. Install

```bash
cd frontend
npm install
```

## 2. Run

```bash
npm run dev
```

Default address: `http://localhost:5173`

## 3. Backend API base url

Optional environment file:

```bash
# frontend/.env
VITE_API_BASE_URL=http://localhost:8000
```

If not configured, default is `http://localhost:8000`.
You can copy `frontend/.env.example` to `frontend/.env`.
